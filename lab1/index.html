  • HTML
  • CSS
  • Python
  • Django
  • JavaScript

De forma a consolidar todos os conhecimentos, será feito um laboratório semanal, de entrega obrigatória, de aplicação da matéria lecionada. Irá ser desenvolvido também um projeto.
